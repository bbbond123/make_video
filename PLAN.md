# AI 视频批量生产系统 - 项目计划

## 项目目标

构建一个自动化视频生产流水线，实现：
- 输入：文案
- 输出：每天 3 条 AI 虚拟人物视频（30-60秒）
- 平台：全平台通用（抖音、小红书、视频号、快手等）

---

## 一、视频形态（三选一或组合）

| 形态 | 描述 | 复杂度 | 适用场景 |
|-----|------|--------|---------|
| AI真人口播 | 固定人物正面说话，口型同步 | 中 | 信任感强，适合观点输出 |
| 人物+场景 | 人物在生活场景中，配旁白 | 中 | 情绪/感悟类，容错率高 |
| 定格+字幕 | 人物轻微动作，重点靠字幕配音 | 低 | 批量最快，稳定 |

**建议**：先从「定格+字幕」或「AI真人口播」起步

---

## 二、人物形象设定

### 待确定项
- [ ] 性别：男 / 女
- [ ] 年龄：25-35（推荐）
- [ ] 风格：干净、普通、不浮夸
- [ ] 身份设定：上班族 / 宝妈 / 城市青年

### 原则
- 固定 1-2 个长期人物
- 不要每条视频换人（影响账号权重）

---

## 三、文案结构规范

```
标题钩子（1句话，吸引注意）
正文（80-150字，口语化）
结尾（共鸣/提问，引导互动）
```

### 示例
```
你有没有发现，
成年人最累的不是加班，
而是明明很努力，
却没人真正理解你。

如果你也有这种感觉，
别急，你不是一个人。
```

## 四、技术方案（待选型）

### 核心模块

| 模块 | 功能 | 候选工具/API |
|-----|------|-------------|
| 人物生成 | 照片→视频 / 文本→口播视频 | HeyGen, D-ID, Synthesia |
| 配音 | 中文自然语音 | 阿里云TTS, 微软Azure, ElevenLabs |
| 字幕 | 自动生成+样式 | Whisper, 剪映API |
| 剪辑合成 | 模板化剪辑 | FFmpeg, MoviePy |
| 批量调度 | 自动化流程 | Python脚本 / n8n / Make |

### 待调研
- [ ] 各平台API定价
- [ ] 中文口型同步效果
- [ ] 是否支持批量调用

## 五、自动化流程设计

```
输入文案
    ↓
文案预处理（分段、时长估算）
    ↓
AI配音生成
    ↓
AI人物视频生成（口型/动作）
    ↓
字幕生成
    ↓
模板剪辑合成
    ↓
导出（9:16，抖音/小红书规格）
```
